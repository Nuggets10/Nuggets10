Hi there! I'm a 16-year-old developer from Italy with a passion for technology.

Most of my work revolves around computer science, specifically using Python and C++. 
I’m fascinated by the world of radio communications and Software Defined Radio (SDR). As of now I like to experiment listening to different radio frequencies using an SDR attached to my laptop.
I'm also interested in the intersection between software and electronics. I sometimes work with microcontrollers like the ESP32 to build useful portable devices.
